## 프로젝트 핵심

- 이 프로젝트는 Spring Boot + Elasticsearch(ECK/K8s) 기반 한글 식품 검색 시스템으로, dense_vector(kNN)와 BM25를 결합한 하이브리드 검색이 핵심입니다.
- 코드 구조는 indexing(버전 인덱스 생성/재색인/alias 전환), search(전략·쿼리빌더·요청빌더·응답매퍼), synonym, embedding, categoryboost로 책임이 분리되어 있습니다.
- docs_tasks 기록상 작업 흐름은 dense_vector(kNN) 검색 → 동의어 동적 반영 → 무중단 인덱스 롤아웃 → 사용자사전/Nori 적용 → 카테고리부스팅 고도화(TTL 캐시·beta·cutoff) → KnnSearchStrategy 리팩터링 순으로 진행됐습니다.
- 운영 포인트는 application.yml 중심 설정(read-alias, min-score-threshold, category-boost-cache-ttl)과 sh_bin/ 스크립트(ECK/사전/검증 자동화)이며, 문서-코드-운영 절차가 연결되어 있습니다.
- 현재 상태는 기능 확장보다 구조 안정화 단계에 가깝고, 테스트도 검색/동의어/카테고리부스팅 중심으로 회귀 검증 체계를 갖춘 형태입니다.
